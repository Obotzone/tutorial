Langkah demi langkah

Guided Setup

Pastikan proyek Cloudflare Pages kamu sudah online

Sebelum menghubungkan subdomain, pastikan kamu sudah punya proyek aktif di Cloudflare Pages. Minimal:

  • Sudah membuat proyek dan deploy dari Git (GitHub/GitLab) atau upload manual.
  • Website bisa dibuka di alamat bawaan seperti namaproject.pages.dev.

Belum punya? Selesaikan dulu Tutorial #1 tentang membuat project pertama di Cloudflare Pages.

Catat alamat default Cloudflare Pages

Buka dashboard Cloudflare ‚Üí pilih menu Pages ‚Üí klik proyek yang ingin kamu hubungkan.

Di bagian atas biasanya tertulis alamat default, misalnya:

  • namaproject.pages.dev

Catat persis alamat ini, karena akan digunakan sebagai target di DNS (CNAME).

Tentukan subdomain yang ingin kamu pakai

Sekarang tentukan nama subdomain yang akan kamu gunakan sebagai alamat resmi website. Contoh:

  • portfolio.dpdns.org
  • projekmu.dpdns.org
  • brandkamu.dpdns.org

Penting: di beberapa penyedia subdomain, kamu bisa memilih banyak ekstensi (misalnya beberapa pola subdomain). Sesuaikan dengan opsi yang tersedia di penyedia yang kamu gunakan.

Buat CNAME record ke Cloudflare Pages

Di sini konsepnya sederhana: subdomain kamu akan diarahkan ke alamat .pages.dev milik proyekmu menggunakan DNS CNAME.

Masuk ke panel DNS tempat subdomain dikelola. Jika kamu mendapat subdomain dari penyedia seperti dpdns.org, biasanya admin mereka yang akan membuat CNAME ini untukmu.

Format umumnya:

  • Type: CNAME
  • Name/Host: bagian depan subdomain (misal: portfolio)
  • Target: alamat Pages kamu, misalnya namaproject.pages.dev
  • TTL: Auto
  • Proxy: On (di Cloudflare biasanya icon oranye)

Kalau kamu adalah admin domain dpdns.org, CNAME dibuat pada zona domain itu. Kalau kamu hanya pengguna subdomain, cukup kirimkan data di atas ke admin/penyedia subdomain.

Tambahkan custom domain di Cloudflare Pages

Setelah DNS diarahkan, kembali ke dashboard Cloudflare Pages:

  • Buka proyek yang sama.
  • Pilih tab Custom domains.
  • Klik Set up a custom domain.
  • Masukkan subdomain lengkap, misalnya: portfolio.dpdns.org.

Cloudflare akan melakukan verifikasi DNS. Jika CNAME sudah benar, status akan berubah menjadi Active dan HTTPS otomatis dibuat.

Proses aktivasi HTTPS bisa butuh beberapa menit. Tunggu sampai status SSL/TLS-nya sudah hijau sebelum kamu tes di browser.

Uji akses & perbaiki kalau belum tersambung

Coba buka subdomain kamu di browser:

  • https://portfolio.dpdns.org

Jika halaman dari Cloudflare Pages muncul, artinya konfigurasi berhasil. Kalau belum:

  • Cek lagi CNAME: apakah sudah mengarah ke namaproject.pages.dev yang benar?
  • Cek ejaan subdomain: tidak ada spasi/karakter asing.
  • Tunggu 5‚Äì10 menit untuk propagasi DNS.

Masih error? Kamu bisa minta admin penyedia subdomain (misalnya penyedia dpdns.org) untuk cek ulang DNS, atau konsultasi langsung via kontak support yang mereka sediakan.

Optimalkan branding & exposure subdomainmu

Setelah subdomain tersambung, jangan berhenti di teknis saja. Agar alamatmu bermanfaat untuk jangka panjang:

  • Pastikan isi websitenya benar-benar membantu orang lain.
  • Update konten secara berkala (bukan sekadar ‚Äúlanding page kosong‚Äù).
  • Bagikan link di media sosial, grup, atau komunitas relevan.
  • Gunakan nama subdomain yang konsisten dengan brand-mu.

Subdomain gratis + konten yang kuat = kombinasi yang bisa tetap mendatangkan trafik organik, apalagi kalau sudah punya exposure dari waktu ke waktu.

Ringkasan singkat

Di tutorial #2 ini kamu belajar:

  • Mengecek proyek aktif di Cloudflare Pages.
  • Menentukan subdomain yang ingin dipakai.
  • Membuat CNAME dari subdomain ke alamat .pages.dev.
  • Mendaftarkan subdomain sebagai Custom Domain di Pages.
  • Testing & troubleshooting dasar jika belum tersambung.

Catatan untuk pengguna subdomain gratis:
Jika kamu mendapatkan subdomain dari penyedia seperti dpdns.org, biasanya kamu tidak perlu menyentuh DNS sendiri. Cukup:

  • Request nama subdomain yang kamu inginkan.
  • Kirimkan alamat namaproject.pages.dev ke admin.
  • Tunggu admin menyetel CNAME di sisi mereka.
