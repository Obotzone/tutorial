Menghubungkan Domain / Subdomain ke Cloudflare Pages

Setelah project kamu online di *.pages.dev, sekarang saatnya membuatnya profesional dengan domain atau subdomain kamu sendiri — termasuk subdomain gratis digitalplat.org.

1. Apa yang akan kita lakukan?

Cloudflare Pages mengizinkan kamu memakai **domain apa saja** selama kamu punya akses DNS-nya. Yang bisa dipakai:

  • Domain pribadi (misal: obotzone.id)
  • Subdomain pribadi (misal: blog.obotzone.id)
  • Subdomain digitalplat.org (misal: nama-user.digitalplat.org)
Intinya: selama kamu bisa mengubah DNS → kamu bisa hubungkan ke Pages.

2. Masuk ke Dashboard Project Pages

Buka:

Workers & Pages → Pages → Pilih project kamu

Setelah masuk project, pilih tab:

  • Custom Domains
  • Klik tombol Set up a custom domain

3. Masukkan domain / subdomain kamu

Masukkan domain atau subdomain yang ingin kamu hubungkan.

Contoh: landing.obotzone.id atau username.digitalplat.org

Kemudian klik Continue.

4. Cloudflare menampilkan DNS Records

Cloudflare Pages akan memberikan 1 record yang HARUS kamu buat:

  • Type: CNAME
  • Name: subdomain kamu
  • Target: nama-project.pages.dev
  • Proxy: ON / Proxied (ikon oranye)
Pastikan mode proxy (awan oranye) aktif. Ini penting agar SSL otomatis aktif.

5. Untuk subdomain digitalplat.org

Jika kamu memakai penyedia subdomain gratis digitalplat.org, DNS editor biasanya berupa:

  • Panel DNS sederhana
  • Mendukung CNAME
  • Pengaturan propagasi cepat

Kamu cukup menambahkan: 

Type: CNAME
Host: nama-kamu
Target: nama-project.pages.dev
Proxy: ON (jika ada)
TTL: Auto
Jangan gunakan A Record. Cloudflare Pages hanya bekerja dengan CNAME.

6. Validasi & aktifkan

Setelah DNS kamu buat, kembali ke Cloudflare Pages lalu klik:

  • Check DNS

Jika sudah sesuai, Cloudflare akan menampilkan tulisan:

Custom domain is now active ✔

7. SSL otomatis aktif

Setelah domain aktif, Cloudflare akan membuat sertifikat SSL otomatis. Biasanya butuh:

  • 1 menit − 15 menit

Kamu akan melihat status:

SSL/TLS: Active

8. Domain kamu siap dipakai

Website kamu sekarang bisa diakses lewat domain atau subdomain yang kamu hubungkan.

Contoh: https://tutorial.obotzone.id
atau https://nama-user.digitalplat.org

Ini jauh lebih profesional untuk portfolio, landing page, atau jaringan exposure Obotzone.
