Pengenalan Cloudflare Pages

Dasar paling penting sebelum masuk ke teknis: memahami konsep Cloudflare Pages, cara kerjanya, apa kelebihannya, dan bagaimana ini mendukung sistem subdomain & exposure Obotzone.

1. Apa itu Cloudflare Pages?

Cloudflare Pages adalah platform hosting gratis untuk website statis (HTML, CSS, JS) yang berjalan di jaringan global Cloudflare. Ia memberikan kecepatan tinggi, SSL otomatis, dan deployment instan.

Alur kerjanya:
Update file → commit → Cloudflare Pages build → website otomatis online.

2. Kenapa Cloudflare Pages cocok untuk Obotzone?

  • Gratis & sangat cepat.
  • Cocok untuk banyak subdomain (termasuk dari digitalplat.org).
  • Deployment otomatis, tanpa upload manual.
  • Stabil untuk website ringan & landing page.

3. Integrasi dengan subdomain digitalplat.org

Subdomain gratis dapat diarahkan ke project Cloudflare Pages. Ini ideal untuk:

  • Membuat banyak landing page.
  • Website eksperimen.
  • Halaman tutorial.
  • Jaringan exposure Obotzone.
Semakin besar exposure Obotzone, semakin bernilai semua subdomain di bawah jaringan situsmu.

4. Apa saja yang perlu disiapkan?

  • Akun Cloudflare.
  • File website statis (HTML + CSS).
  • Akses DNS subdomain digitalplat.org.
